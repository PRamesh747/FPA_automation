# HSBC FP&A Financial Analysis Automation

## Project Overview
This project automates key FP&A financial ratio calculations for HSBC Bank using Python.  
It replaces manual Excel-based analysis with a reproducible and structured workflow.

The project uses publicly available annual report data and focuses on accuracy, clarity, and professional reporting.

---

## Objectives
- Automate FP&A ratio calculations
- Reduce manual Excel work
- Create clean, reusable financial analysis outputs
- Demonstrate real-world banking FP&A skills

---

## Financial Metrics Calculated
- Return on Equity (ROE)
- Return on Assets (ROA)
- Cost-to-Income Ratio
- Equity Ratio

## Workflow Summary
1. Extracted key financial figures from HSBC Annual Report
2. Structured raw data into CSV format
3. Cleaned and validated financial data using Python
4. Calculated FP&A ratios programmatically
5. Exported results to a formatted Excel report
